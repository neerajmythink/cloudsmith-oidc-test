# My Sample Package

A simple Python project created to demonstrate building and pushing to Cloudsmith using GitHub Actions and OIDC.

## Features
* Standardized `pyproject.toml` layout.
* Ready for automated CI/CD workflows.
* Supports OIDC-based authentication for secure uploads.

## Building the Package
To build the package locally, run:
```bash
python -m build
```
This will create distribution files in the `dist/` directory.
